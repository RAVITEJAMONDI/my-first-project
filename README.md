# Python Mini Projects

This repository contains beginner-friendly Python projects developed to improve programming and problem-solving skills.

## Projects Included
- Simple Calculator
- Password Generator
- To-Do List Application

## Skills Used
- Python
- File Handling
- Problem Solving

## How to Run

1. Install Python
2. Download or clone this repository
3. Run any file using:

python calculator.py  
python password_generator.py  
python todo.py
